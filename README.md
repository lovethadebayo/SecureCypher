🔐 SecureCypher

SecureCypher is a powerful, web-based encryption tool that allows users to encrypt and decrypt text and files using multiple industry-standard algorithms.

Features

- Multiple Encryption Algorithms: AES-256-CBC, AES-128-CBC, DES, 3DES, Blowfish, and RSA
- Text Encryption: Encrypt and decrypt any text message
- File Encryption: Upload and encrypt any file format (.txt, .pdf, .jpg, .docx, etc.)
- Password Visibility Toggle: Show/hide password as you type
- Copy to Clipboard: Easy copy of results with one click
- Save as File: Export encrypted/decrypted text as a downloadable file
- RSA Key Generation: Generate public/private key pairs for asymmetric encryption
- Responsive Design: Works on desktop, tablet, and mobile

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- CryptoJS Library
- JSEncrypt Library

📁How to Use

Text Mode
1. Enter your text in the input box
2. Choose an encryption key (password)
3. Select your preferred algorithm
4. Click "Encrypt" to encrypt or "Decrypt" to decrypt
5. Copy or save the result

File Mode
1. Switch to File Mode tab
2. Upload any file
3. Enter encryption key
4. Select algorithm
5. Click "Encrypt File" or "Decrypt File"

🔒 Security Note

- Your encryption keys are never stored or transmitted
- All encryption happens locally in your browser
- For maximum security, use strong, unique passwords
